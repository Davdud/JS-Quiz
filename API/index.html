Getting Started...

We have created an API for you webapps. This is live and responsive data that can update from time to time when our team chooses to fix bugs, add more content or when updating the syle of the dynamic page. Here's how you can work with our code!

How to link our api


You don't need an API key to use our data as we have chosen to make it as smooth and easy for you as possible! Just copy paste this code into your web apps (paste this at the beginning of your Javascript):

fetch('https://gkquizapp.netlify.app/api/api.json')

Now you can do all sorts of things with it! Eg:

fetch('https://gkquizapp.netlify.app/api/api.json')
.then(res -> console.log(res))

Your web app code should look something like this...

APP.js -->
const data = fetch('https://gkquizapp.netlify.app/api/api.json')
console.log(data)
